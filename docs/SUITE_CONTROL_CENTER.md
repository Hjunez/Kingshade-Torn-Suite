# Suite Control Center

Suite Control Center is hosted by Scout and coordinates Scout and War Tools. It is available from the KS button on faction member-list pages.

## Tabs

- **Overview:** component/version health, API-key and consent state, status age/member count and local-data counts.
- **Scout:** API key, mandatory disclosure/consent, Scout preferences and maintenance actions.
- **War Tools:** persistent filter/sort/threshold/collapse settings.
- **Data:** privacy details and targeted cache clearing.

## Safety and compliance

- Network requests remain disabled until the API disclosure is accepted.
- Requests pause whenever Torn is hidden or unfocused.
- The Suite uses only the manually opened page, Torn's official API and FFScouter's documented endpoint.
- The Suite changes local presentation and data settings only; it performs no Torn actions.
