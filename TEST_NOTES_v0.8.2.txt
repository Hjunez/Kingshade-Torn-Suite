Kingshade Suite 0.8.2 test package

Install both files together:
- Kingshade_Scout_Torn_PDA_v0.7.0_FINAL.user.js
- KS_War_Tools_Torn_PDA.user.js

Both components must display version 0.8.2.

Changes:
- Traveling members without a trustworthy ETA now show TRAVEL and ~? instead of a bare Traveling label.
- Tapping ~? explains whether the destination, route time, departure estimate, or prior ETA is unavailable.
- Status labels and timers are forced onto separate centered rows on narrow mobile screens.
- Timer summary now separates usable travel estimates from unknown travel ETAs.
- Abroad remains without a countdown because the player is stationary abroad.
- Hospital/Jail/Federal countdowns remain exact.
- Attacker names remain clickable only when Torn exposes a verifiable player profile ID.
