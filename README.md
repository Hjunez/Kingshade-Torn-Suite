# 🎬 Kingshade's Bootlegging

> 📌 This script highlights the next action to take for the most efficient Bootlegging strategy.

---

# 🔵 Blue Highlight - COPY

When a genre is highlighted in **blue**, keep copying that genre.

✅ Continue until the highlight changes or disappears.

---

# 🟢 Green Highlight - SELL

When the **Sell Counterfeit DVDs** button is highlighted in **green**:

✅ Click **Sell Counterfeit DVDs**.

---

# ⚫ No Highlight

If there is **no highlight**, do nothing.

⏳ The script has determined that your current queue is already sufficient.

Wait until the queue becomes smaller. The script will automatically highlight the next action when needed.

---

# 📖 How to Use

1. Open Bootlegging.
2. 🔵 Blue = Copy.
3. 🟢 Green = Sell.
4. ⚫ No highlight = Wait.
5. Repeat.
